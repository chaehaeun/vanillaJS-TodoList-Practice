## 간단한 투두리스트 구현

---

- ### 구현하고 싶은 기능

  1. 리스트 추가
  2. 리스트 삭제
  3. 로컬스토리지를 사용해 새로고침해도 리스트가 남아있게 하기
  4. 삭제시 로컬스토리지내의 자료도 삭제 될 수 있게
  5. 체크박스 기능

  1. 로컬스토리지 안의 자료가 삭제한 버튼의 인덱스와 따로 놀며 삭제되는 중...혼자서 해보려니 원인 파악 쉽지 않다.
